C Programming Internship — Solutions

This folder contains reference implementations for the C-language tasks from the internship sheet.

Included sample programs:
- `task1_calculator.c` — Basic calculator using `switch` for +, -, *, /
- `task2_matrix.c` — Matrix operations: addition, multiplication, transpose using 2D arrays and functions
- `task3_student_mgmt.c` — Menu-driven student management system using `struct` and file handling

How to compile (macOS / Linux):
```bash
gcc -std=c11 -O2 -Wall "C Programming internship/solutions/task1_calculator.c" -o "C Programming internship/solutions/task1_calculator"
gcc -std=c11 -O2 -Wall "C Programming internship/solutions/task2_matrix.c" -o "C Programming internship/solutions/task2_matrix"
gcc -std=c11 -O2 -Wall "C Programming internship/solutions/task3_student_mgmt.c" -o "C Programming internship/solutions/task3_student_mgmt"
```

Run examples:
```bash
./"C Programming internship/solutions/task1_calculator"
./"C Programming internship/solutions/task2_matrix"
./"C Programming internship/solutions/task3_student_mgmt"
```
